Instruction:

There contains the Visual Studio files for Windows users.
You should install Visual Studio IDE in advance.
Open the Visual Studio Solution file "a0.sln" using  Visual Studio IDE,
and you can edit "main.cpp" and build the executive file "a0.exe".
We tested it in Windows 10 and Visual Studio 2019 (toolset v142).
If you are using a newer version VS, please retarget the project according to your platform.

*****************************************************

Reference Solution:

a0soln.exe is a reference solution executive file.
You can check it in command, for example run the command:
a0soln.exe < torus.obj
